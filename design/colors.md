
## Usage rules

- Основной фон — глубокий navy.
- Акцентный желтый использовать точечно: кнопки, важные маркеры, детали логотипа, свет фонаря.
- Не использовать красный как основной цвет блока рисков. Риски объяснять спокойно, без тревожной визуальной агрессии.
- Карточки должны быть темнее/светлее фона только на 1–2 шага, без резкого контраста.
- Границы тонкие, почти незаметные.
