# SEO and Accessibility Requirements

## Meta

Title:

```text
microcredit.kz — микрокредиты в Казахстане простым языком
```

Description:

```text
Независимый информационный портал о микрокредитах в Казахстане: риски, права заемщика, договоры, просрочка и финансовая грамотность без рекламы займов.
```

## Semantic HTML

- Использовать один `h1` на странице.
- Секции оформлять через `section`.
- Навигацию через `nav`.
- Footer через `footer`.
- Кнопки должны быть настоящими `button` или ссылками `a`, если ведут к разделу.

## Accessibility

- Контраст текста должен быть достаточным.
- Все интерактивные элементы должны иметь focus state.
- Навигация должна быть доступна с клавиатуры.
- Изображения должны иметь осмысленный `alt`, если они контентные.
- Декоративные изображения можно скрывать через `aria-hidden="true"`.

## Performance

- Оптимизировать изображения.
- Не использовать огромные изображения без необходимости.
- Hero должен адаптироваться под мобильные устройства.
- На мобильной версии декоративный фон не должен ломать читаемость.
