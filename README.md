# microcredit.kz — Codex Starter Brief

Этот пакет нужен, чтобы Codex сделал первую качественную версию сайта с минимальным количеством лишних итераций.

## Как использовать

1. Создай или открой проект React + Vite + TypeScript.
2. Скопируй папки `design/` и `assets/` в корень проекта.
3. Положи свои файлы логотипа в `assets/`:
   - `logo.png`
   - `favicon.png`
   - `apple-touch-icon.png` — можно временно сделать копией `favicon.png` или `logo.png`, лучше размер 180×180.
4. Открой `codex-prompt.md`.
5. Вставь весь текст из `codex-prompt.md` в Codex.

## Важно

- Не проси Codex “придумать сайт заново”.
- Пусть он строго следует файлам из папки `design/`.
- `hero-reference.jpg` — это референс атмосферы, а не обязательная финальная картинка.
- Если логотип уже есть, Codex не должен генерировать новый.
